# editm

editm is a simple markdown editor.

## Usage

### Clone

```bash
git clone https://github.com/MKAbuMattar/editm.git
```

### Install

```bash
cd editm

yarn install
```

### Run

```bash
yarn dev
```

### Build

```bash
yarn build
```
